# Bases técnicas de fase

Esta pasta é interna ao pipeline e não faz parte da publicação do site.

Os arquivos PNG são gerados automaticamente, revisados tecnicamente e aprovados
por uma pessoa antes de servirem como referência binária para Azul, Rosa e
Arco-íris. O repositório mantém apenas este arquivo de orientação; as imagens e
o estado operacional ficam ignorados pelo Git até uma decisão explícita.
