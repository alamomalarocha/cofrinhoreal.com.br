# Checkpoint - Reestrutura multipaginas 001

Data: 2026-07-02

## Objetivo

Registrar a mudanca do site Cofrinho Real de uma landing page unica e longa para um prototipo visual estatico multipaginas.

## Estrutura atual

- `index.html`: Home curta de apresentacao principal.
- `o-que-e.html`: explicacao completa do Cofrinho Real.
- `como-funciona.html`: fluxo conceitual de funcionamento.
- `comerciantes.html`: pagina para comerciantes / Pessoa Juridica.
- `familias.html`: pagina para familias / Pessoa Fisica.
- `educacao.html`: educacao financeira com o Pig.
- `jogos.html`: jogos educativos futuros.
- `pig-coins.html`: PigCoins como pontos educativos simbolicos.
- `faq.html`: perguntas frequentes.
- `privacidade.html`, `termos.html`, `cookies.html`, `direitos.html`: paginas legais estaticas.

## Decisoes

- A Home deve mostrar apenas o essencial: problema, proposta, PF/PJ, mockup, resumo e CTAs.
- O menu agora aponta para paginas reais, nao apenas secoes da Home.
- O projeto continua com `noindex`.
- Header, rodape e identidade visual devem permanecer consistentes.
- Login, busca, App Store/Google Play, Conta PIG e PIX continuam apenas visuais.

## Limites

Esta versao continua sem backend, sem banco de dados, sem login real, sem autenticacao real, sem PIX real, sem app real, sem checkout e sem movimentacao real de dinheiro.

## Proximos passos sugeridos

- Revisar copy da Home mais curta.
- Revisar consistencia visual das paginas internas.
- Evoluir personagens da Vila Pig somente depois da arquitetura visual estar estavel.
