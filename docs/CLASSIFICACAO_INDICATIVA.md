# Classificação indicativa

Status: pesquisa. O mapeamento local ainda aguarda revisão profissional.

Cada conteúdo futuro deve registrar nivel interno, sistema local, descritores, pais, supervisão, status e data de revisão. Tags incluem medo, violencia, linguagem, compras, interacao, dados, publicidade, risco, criptografia e segurança.

No Brasil, jogos e aplicativos possuem orientacoes proprias e podem usar classificação conforme os criterios oficiais: [Guia de Classificação](https://www.gov.br/mj/pt-br/assuntos/seus-direitos/classificacao-1/paginas-classificacao-indicativa/guia-de-classificacao) e [Jogos e aplicativos](https://www.gov.br/mj/pt-br/assuntos/seus-direitos/classificacao-1/paginas-classificacao-indicativa/jogos-e-apps).

Classificação de conteúdo e aferição de idade são mecanismos diferentes: [explicação do MJSP](https://www.gov.br/mj/pt-br/assuntos/noticias/entenda-a-diferenca-entre-classificacao-indicativa-e-afericao-de-idade-1).
