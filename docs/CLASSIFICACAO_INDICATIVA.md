# Classificacao indicativa

Status: pesquisa. O mapeamento local ainda aguarda revisao profissional.

Cada conteudo futuro deve registrar nivel interno, sistema local, descritores, pais, supervisao, status e data de revisao. Tags incluem medo, violencia, linguagem, compras, interacao, dados, publicidade, risco, criptografia e seguranca.

No Brasil, jogos e aplicativos possuem orientacoes proprias e podem usar classificacao conforme os criterios oficiais: [Guia de Classificacao](https://www.gov.br/mj/pt-br/assuntos/seus-direitos/classificacao-1/paginas-classificacao-indicativa/guia-de-classificacao) e [Jogos e aplicativos](https://www.gov.br/mj/pt-br/assuntos/seus-direitos/classificacao-1/paginas-classificacao-indicativa/jogos-e-apps).

Classificacao de conteudo e afericao de idade sao mecanismos diferentes: [explicacao do MJSP](https://www.gov.br/mj/pt-br/assuntos/noticias/entenda-a-diferenca-entre-classificacao-indicativa-e-afericao-de-idade-1).
