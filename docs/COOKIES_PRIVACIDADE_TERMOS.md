# Cookies, Privacidade, Termos e Direitos

## Estado atual

O Cofrinho Real esta em fase de protótipo visual estático.

Não existe:

- backend;
- banco de dados;
- login real;
- cadastro real;
- autenticação;
- PIX real;
- app publicado;
- movimentação real de dinheiro;
- coleta real de dados financeiros.

## Páginas criadas

- `privacidade.html`
- `termos.html`
- `cookies.html`
- `direitos.html`

Essas páginas são documentos iniciais para protótipo visual. Elas não substituem revisão jurídica.

## Cookies e armazenamento local

O site não usa cookies de rastreamento, publicidade ou analytics.

O aviso de privacidade usa `localStorage` apenas para lembrar que o visitante fechou o aviso. Esse dado fica no navegador e não e enviado para servidor.

## Regras futuras

Antes de qualquer operação real, será necessário:

- revisar documentos juridicos;
- definir política de privacidade completa;
- definir termos de uso reais;
- definir canais para direitos do titular;
- revisar tratamento de dados de crianças;
- revisar segurança antes de qualquer PIX ou dinheiro real.

## Linguagem obrigatória

Enquanto for protótipo:

- "Protótipo visual em validação."
- "Sem login real."
- "Sem PIX real."
- "Sem movimentação real de dinheiro."
- "Documento inicial para protótipo visual."

Nunca afirmar que o produto esta juridicamente pronto ou operacional.
