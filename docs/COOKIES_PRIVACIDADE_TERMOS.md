# Cookies, Privacidade, Termos e Direitos

## Estado atual

O Cofrinho Real esta em fase de prototipo visual estatico.

Nao existe:

- backend;
- banco de dados;
- login real;
- cadastro real;
- autenticacao;
- PIX real;
- app publicado;
- movimentacao real de dinheiro;
- coleta real de dados financeiros.

## Paginas criadas

- `privacidade.html`
- `termos.html`
- `cookies.html`
- `direitos.html`

Essas paginas sao documentos iniciais para prototipo visual. Elas nao substituem revisao juridica.

## Cookies e armazenamento local

O site nao usa cookies de rastreamento, publicidade ou analytics.

O aviso de privacidade usa `localStorage` apenas para lembrar que o visitante fechou o aviso. Esse dado fica no navegador e nao e enviado para servidor.

## Regras futuras

Antes de qualquer operacao real, sera necessario:

- revisar documentos juridicos;
- definir politica de privacidade completa;
- definir termos de uso reais;
- definir canais para direitos do titular;
- revisar tratamento de dados de criancas;
- revisar seguranca antes de qualquer PIX ou dinheiro real.

## Linguagem obrigatoria

Enquanto for prototipo:

- "Protótipo visual em validação."
- "Sem login real."
- "Sem PIX real."
- "Sem movimentação real de dinheiro."
- "Documento inicial para protótipo visual."

Nunca afirmar que o produto esta juridicamente pronto ou operacional.
