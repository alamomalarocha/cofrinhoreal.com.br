# Educacao Financeira

O Cofrinho Real ensina educacao financeira sem culpa, medo ou julgamento.

## Principios

- Ensinar planejamento.
- Incentivar sonhos.
- Mostrar consequencias com carinho.
- Ajudar familias.
- Respeitar a idade do usuario.
- Ensinar a guardar sem incentivar avareza.
- Ensinar a gastar com consciencia, sem culpa.
- Permitir doacao futura sempre como escolha, nunca como pressao.

## Linguagem

Usar frases positivas:

- "Seu sonho ficou mais perto."
- "Quer pensar um pouco antes de gastar?"
- "Guardar tambem e uma escolha."

Evitar:

- culpa
- vergonha
- ameaca
- pressao
- comparacao com outras pessoas

## No prototipo visual

O site pode apresentar conceitos, frases e exemplos visuais.

Nao deve criar:

- pontuacao real;
- desafio viciante;
- ranking financeiro;
- recompensa financeira;
- promessa de ganho;
- coleta de dados de criancas.
