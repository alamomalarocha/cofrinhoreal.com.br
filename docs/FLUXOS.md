# Fluxos

Este documento descreve fluxos conceituais. Nao representa implementacao tecnica.

## Receber troco no comerciante

1. Comerciante informa o valor do troco.
2. Comerciante pergunta se o cliente prefere receber no Cofrinho Real.
3. Cliente apresenta QR Code ou numero do cofrinho.
4. Comerciante envia o Pig.
5. Cliente recebe confirmacao.

## Enviar para outra pessoa

1. Usuario escolhe contato, QR Code ou numero do cofrinho.
2. Informa valor.
3. Confirma envio.
4. Recebedor recebe notificacao.

## Conta infantil

1. Responsavel cria ou vincula conta.
2. Define permissoes.
3. Acompanha movimentacoes.
4. Crianca interage com Pig de forma educativa.

## Reflexao antes de gastar

1. Usuario tenta usar saldo.
2. Sistema mostra mensagem positiva.
3. Usuario decide continuar ou guardar.
4. Nao ha julgamento nem bloqueio moral.
