como funciona

Um fluxo simples para testar a ideia.

Neste protótipo, tudo é visual e conceitual. Nenhuma transação real é feita. A ideia é validar se o fluxo faz sentido antes de construir qualquer sistema.

1

Na hora do troco, ele pergunta: “Você prefere receber no Cofrinho Real?”

2

O cliente recebe

O valor entra no cofrinho como um Pig, usando QR Code ou número do cofrinho no futuro.

3

O dinheiro ganha propósito

O usuário pode guardar, enviar para a família ou usar em outro estabelecimento em versões futuras.

faq

Perguntas rápidas para validar a proposta.

O Cofrinho Real é um banco?

Não. A proposta é ser uma plataforma de troco digital e educação financeira.

Este protótipo movimenta dinheiro real?

Não. Esta fase é apenas visual, estática e sem integrações financeiras.

Já existe login, PIX ou conta real?

Não. O objetivo agora é validar a identidade e a proposta antes de criar sistema.

Pig Coin vale dinheiro?

Não. Pig Coin é simbólica e serve apenas para recompensas educativas e visuais.

O projeto incentiva consumo?

Não. O Cofrinho Real incentiva planejamento, reflexão, família e solidariedade.
