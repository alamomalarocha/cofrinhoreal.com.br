O Brasileirinho lembra:

Código de usuário nunca entra automaticamente em produção. Toda proposta passa por revisão humana, técnica e de segurança.

Sempre ativo: sem login, backend, envio, JavaScript de usuário, carteira real ou acesso a serviços externos.

Funciona agora

Desafios locais, HTML e CSS sanitizados, preview isolado, resumo e download local.

Não funciona

Conta, chat, votação, carteira persistente, GitHub, Cloudflare, Codex ou publicação.

Exige projeto futuro

Backend privado, autenticação, moderação, auditoria, limites de custo e aprovação formal.
